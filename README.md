# hubexplorer

### Installation
```
pip3 install .
hubexplorer completion install
```

### Usage
```
usage: hubexplorer [-h] [-v VERSION]
                   {repo,r,login,l,createrepo,c,completion} ...

Easy github explorer

optional arguments:
  -h, --help            show this help message and exit
  -v VERSION, --version VERSION
                        Show version

Sub commands:
  {repo,r,login,l,createrepo,c,completion}
    repo (r)
    login (l)
    createrepo (c)
    completion          Bash auto completion using argcomplete python package.
```
